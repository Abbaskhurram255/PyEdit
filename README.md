# PyEdit
A simple python editor for linux using python and
pygobject *Gtk3*. 


Features 
========
------
- Syntax hilighting 
- code suggestions
- Backup file creation
- Auto saving 
- Auto close bracket an quotes. 


Dependencies 
============
------------
- Python3 and above
- Jedi 
- Gtk3
